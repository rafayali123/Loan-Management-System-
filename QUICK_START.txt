## ‚ùå IF SOMETHING GOES WRONG

### "Cannot connect to MongoDB"
- Check .env file has correct URI
- Check MongoDB user credentials are correct
- Verify IP whitelist in MongoDB Atlas

### "Login fails"
- Make sure admin user was inserted
- Verify collection name is `admins`
- Check database name is `loan_management_db`

### "Port 8080 already in use"
```powershell
netstat -ano | findstr :8080
taskkill /PID <number> /F
```

### "Application won't start"
- Check if old app is still running
- Try stopping it completely with multiple Ctrl+C presses
- Check logs in terminal for error messages
